# Kvasir

   Dependencies:  
  
   -CMake 3.9  
      get it at: https://cmake.org/download/
      (if you have an older version it's advisable to unistall it first)  
   -OpenCV 2  
      try: apt-get install libopencv-dev  
   -Boost Libraries  
      try: apt-get install libboost-all-dev  
   -Qt5  
      try: apt-get install qttools5-dev
